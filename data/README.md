# Data Sources

All data used in this project are drawn from publicly available sources. No raw data from paid or restricted providers is included in this repository.

---

## Sources

### FBref (via worldfootballR)
- **What:** Match-level Premier League results including goals scored and expected goals (xG) for home and away teams.
- **Seasons:** 2017/18 – 2024/25.
- **How to get it:** Use the `worldfootballR` R package (`fb_match_results()`) or download directly from [fbref.com](https://fbref.com). The player-level season statistics in `data/player/` were obtained via `worldfootballR::load_fb_big5_advanced_season_stats()`.
- **Licence:** FBref data is publicly available for non-commercial use.

### Football-Data.co.uk
- **What:** Pre-match bookmaker odds (Bet365, Pinnacle, Best Available) for EPL fixtures, 2017/18 – 2024/25. One CSV per season, named `E0_YYZZ.csv`.
- **Location in repo:** `data/football-data/`
- **How to get it:** Download free season files from [football-data.co.uk/englandm.php](https://www.football-data.co.uk/englandm.php).
- **Licence:** Freely redistributable for non-commercial purposes.

### ClubElo
- **What:** Pre-match Elo ratings for European club teams, updated after every match.
- **Location in repo:** `data/elo/clubelo_all.csv`
- **How to get it:** Bulk download from [clubelo.com/API](http://clubelo.com/API).
- **Licence:** Publicly available; credit ClubElo.com when using.

### Player statistics (FBref via worldfootballR)
- **What:** Season-level player standard stats (goals, assists, xG, xAG, minutes played) aggregated to team level to construct a player-quality proxy (`team_quality`).
- **Location in repo:** `data/player/fbref_player_2425.csv` (2024/25 season example).
- **How to get it:** `worldfootballR::load_fb_big5_advanced_season_stats(season_end_year, stat_type = "standard", team_or_player = "player")`.

## Processed files (`data/processed/`)

These files are derived from the raw sources above and are included for convenience. Both can be regenerated by running `odds/update_odds.R`.

### `epl_matches_with_xg_odds.csv`
The main modelling dataset. One row per Premier League match.

| Column | Description |
|--------|-------------|
| `season_end_year` | Season identifier (e.g. 2024 = 2023/24 season) |
| `date` | Match date (YYYY-MM-DD) |
| `home_team` | Home team name |
| `away_team` | Away team name |
| `home_goals` | Full-time home goals |
| `away_goals` | Full-time away goals |
| `home_xg` | Home team expected goals (FBref) |
| `away_xg` | Away team expected goals (FBref) |
| `b365h` / `b365d` / `b365a` | Bet365 decimal odds (home / draw / away) |
| `bbavh` / `bbavd` / `bbava` | Best-available average decimal odds |
| `psh` / `psd` / `psa` | Pinnacle decimal odds |
| `o_h` / `o_d` / `o_a` | Chosen best odds (used in profit simulation) |
| `imp_h_raw` / `imp_d_raw` / `imp_a_raw` | Raw implied probabilities (1 / decimal odds) |
| `overround` | Sum of raw implied probabilities (bookmaker margin) |
| `p_home` / `p_draw` / `p_away` | De-vigged implied probabilities (sum to 1) |

### `epl_fbref_with_odds.csv`
Intermediate file: FBref match results joined to Football-Data odds, before final cleaning. Columns are a subset of the above (no xG; used as a merge checkpoint in `odds/update_odds.R`).

## Notes on look-ahead bias

All features derived from these data (rolling form windows, Elo ratings, implied probabilities) are constructed using only information available **before** the match being predicted. The Elo merge uses a rolling join (`data.table::roll = Inf`) to pick the most recent rating on or before the match date.
